# Fitness Pal Landing Page

A lightweight static landing page for **Fitness Pal** with theme persistence and guest/registered navigation state via `localStorage`.

## Run on localhost

### Option 1 (recommended, Node.js)

```bash
npm start
```

Then open:

- <http://localhost:8080>

### Option 2 (Python)

```bash
python3 -m http.server 8080
```

Then open:

- <http://localhost:8080>

## Features

- Light/dark mode toggle with persistence (`localStorage.theme`)
- CTA flows with persistence (`localStorage.userType`)
- Responsive layout (mobile/tablet/desktop)
- Minimal fitness-themed illustration and subtle animations
